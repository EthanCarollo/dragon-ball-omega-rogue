# Dragon Ball: Ruby Edition

Petit jeu en Ruby inspiré de **Dragon Ball** et développé avec **Gosu**.

## Installation

1. Clonez le repo :
```bash
git clone https://github.com/EthanCarollo/dragonball-idk.git
cd dragonball-idk
```
2. Installez **Gosu** :
```bash
gem install gosu
```
3. Lancez le jeu :
```bash
ruby main.rb
```
